# 清除项目初始化配置计划

## 目标
清除WXT + React模板生成的默认配置，使项目回到干净状态。

## 具体步骤

### 1. 清理弹出页面配置
- 删除 `entrypoints/popup/App.css` 和 `entrypoints/popup/style.css` 中的默认样式
- 简化 `entrypoints/popup/App.tsx` 为基础结构
- 保持 `entrypoints/popup/main.tsx` 基础结构但移除默认样式引用
- 清理 `entrypoints/popup/index.html` 中的默认配置

### 2. 清理脚本文件
- 简化 `entrypoints/background.ts` 为基础结构
- 简化 `entrypoints/content.ts` 为基础结构，移除默认的google.com匹配规则

### 3. 清理资源文件
- 移除 `assets/react.svg` 等默认资源
- 清理 `public/icon/` 中的默认图标文件
- 移除 `public/wxt.svg`

### 4. 更新配置文件
- 保持 `package.json` 基础结构但更新项目名称和描述
- 保持 `wxt.config.ts` 基础结构
- 保持 `tsconfig.json` 不变
- 保持 `.gitignore` 不变

## 预期结果
项目将保持基本的WXT + React框架结构，但移除所有默认的示例代码和资源，为后续的开发做好准备。
